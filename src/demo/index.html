Relative chroma palette

The palette's swatches are made with the new notation: oklch(colorspace L RC H), where RC is a relative chroma. You can see that we got nice results compared to next example wit ha fixed absolute chroma value for all swatches.

Absolute chroma palette

Here, we use the same absolute chroma value for all swatches. You can see that for some colors, because of the browser clamping method, we got hue and lightness shifts.
